# Akash-API-Data-Viewer


A modern, responsive, and animated web application built with **HTML, CSS, and JavaScript** that fetches user data from a public API and displays it in a clean, card-based UI.  
This project demonstrates **Fetch API**, **asynchronous JavaScript**, **error handling**, **JSON parsing**, and **responsive design**.
